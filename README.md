# 🧬 E3-ome analyses extensiion

This repository contains additional analyses and companion resources built upon the human E3-ome.

The original E3-ome resource is maintained by the Feltham Laboratory:

**Main E3-ome repository:**
https://github.com/FelthamLaboratory/E3-ome

This repository serves as an extension of the E3-ome project, providing additional datasets, expression atlases, and exploratory analyses of the human E3 ligase landscape.

---

## 🚀 Web application

Explore the original E3-ome web application:

**E3-ome web app:**
https://shiny.wehi.edu.au/chua.n/E3app/

Features:

* Search for specific E3 ligases
* Filter ligases by classification (e.g. RING, HECT, RBR)
* Explore curated annotations and functional information
* Browse the human E3 ligase landscape

---

## 📊 Expression atlases

### 🛡️ Monaco immune cell atlas

Expression profiles of 672 human E3 ligases across immune cell populations from the Monaco dataset.

**Interactive atlas:**
https://jake-c-26.github.io/E3-ome/Expression/Monaco/

The atlas provides:

* Cell type-specific E3 expression profiles
* Immune cell enrichment signatures
* Searchable E3 ligase expression patterns
* Interactive exploration of immune cell populations

Additional atlases will be added over time, including GTEx, Human Protein Atlas and single-cell expression resources.
